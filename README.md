# Decodelab_1

To-do list

A simple and responsive Todo List Web Application built using HTML, CSS, JavaScript, and Python Flask. This project allows users to manage their daily tasks with a clean and user-friendly interface. Task data is stored in a JSON file, so no database setup is required.

Technologies Used:

Frontend:
  - HTML
  - CSS3
  - JavaScript
    
Backend:
  - Python
  - Flask
  - Flask-CORS
    
Data Storage:
  - JSON File (tasks.json)
